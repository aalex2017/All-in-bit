All logical operators, comparison operators, arithmetic operators
(except assignment, unary plus and unary minus) and a couple more
functions are written using bitwise operators.

Все логические операторы, операторы сравнения, арифметические операторы
(кроме присваивания и унарного плюса) и ещё пару функций
записаны при помощи битовых операторов.
